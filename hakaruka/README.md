# FANモジュール重量測定システム

## 概要
FANモジュールの重さを測定し、割り込まれた数を確認してDBに記録するアプリケーションです。

## 機能
- シリアル通信による重量計との連携
- 重量の自動判定（OK/NG）
- データベースへの記録
- 3種類の測定単位（段ボール1箱、1シート、部品1つ）

## データベース構造

### テーブル名: ループ

| カラム名 | 型 | 説明 |
|---------|------|------|
| Serial | NVARCHAR(50) | シリアル番号 |
| Unit | NVARCHAR(50) | 測定単位 |
| Result | NVARCHAR(10) | 判定結果（OK/NG） |
| Weight | DECIMAL(10,2) | 測定重量（グラム） |
| reg_time | DATETIME | 登録日時 |

### SQL作成スクリプト
```sql
CREATE DATABASE hakaruka;
GO

USE hakaruka;
GO

CREATE TABLE ループ (
    ID INT IDENTITY(1,1) PRIMARY KEY,
    Serial NVARCHAR(50) NOT NULL,
    Unit NVARCHAR(50) NOT NULL,
    Result NVARCHAR(10) NOT NULL,
    Weight DECIMAL(10,2) NOT NULL,
    reg_time DATETIME NOT NULL
);
```

## 設定ファイル (config.ini)

```ini
[Settings]
Threshold1=15    ; 段ボール1箱の閾値（グラム）
Threshold2=400   ; 1シートの閾値（グラム）
Threshold3=416   ; 部品1つの閾値（グラム）

[Serial]
ComPort=COM1  ; シリアルポート
BaudRate=9600    ; ボーレート

[Database]
ConnectionString=Server=localhost;Database=hakaruka;Integrated Security=true;
```

## 使用方法

### 処理フロー

1. **アプリ起動**
   - 重量計との接続確認
   - BOXなどのシリアル読み込み（ENTERキー）

2. **測定開始**
   - 重量計を0表示にセット
   - できたら「重さをはかってください」表示（グレー）+ 重量計監視

3. **重量測定**
   - 重さが入ると重量計と通信で確認し、そのあとの数字を取る
   - ComplcSImf.exeを使えばとれるはず

4. **判定処理**
   - 取得した重量が既定の範囲内かを判定
   - 判定結果の不一致が多々あるので注意

5. **データ保存**
   - BOX+NO+単位+判定+時間をDBにinsertする

6. **成功表示**
   - 成功したら事前ヘルプ
   - 重量計にモノがなくなったら0表示にしておく

## UI説明

### 入力フォーム
- **シリアル番号**: 段ボールの重さを記入
- **測定単位**: 1つの段ボールに400個、20個入りのシート×20個で構成
  - 段ボールの重さ = シートの重さ + 記品の重さ
  - 段階的なるOKを測る
  - 外だらNGを出し、許容値数の不一致を是正

### 表示
- **重量計**: 大きく一つのテキストBOX
- **判定**: 重量計が許算値でなければOK、背景いグレー
- **OK表示**: 背景が白に変わり黒の文字でOK
- **NG表示**: 背景が赤でエラーとしたい

### ボタン
- **スキャン/測定**: シリアル番号を入力後、重量を測定
- **保存**: 測定結果をデータベースに保存
- **リセット**: フォームをリセット

## 注意事項

### 調値
調値は測る単位によってかえたい。
単位は以上で切り替えるように：

1. 段ボール1箱: 15以下
2. 1シート: 400以下
3. 部品1つ: 416以下

### 重量計通信
- comPlcSImpF.exeのexeを使って重量計とシリアル通信でデータのやり取りをする
- このexeのコードがなかったら後ほど提供できる
- そして重量計にはすでに使えるコマンドがいくつかあるのでそれも活用する

### ゼロ表示
ゼロ表示のコマンドは重量計専用である。

### 調値は測る単位によってかえたい
単位は以上で切り替えるようにしたい。そして、単位は3種類

1. 段ボール1箱
2. 1シート
3. 部品1つ

### iniファイルの内容
調値1: 15<<16
調値2: 400<<416
...

などで所属した重量が既定の範囲内なのか判定する。

## トラブルシューティング

### シリアルポートが見つからない
- デバイスマネージャーでCOMポートを確認
- config.iniのComPortを正しいポートに変更

### データベース接続エラー
- SQL Serverが起動しているか確認
- ConnectionStringが正しいか確認
- データベースとテーブルが作成されているか確認

### 重量計が反応しない
- シリアルケーブルの接続を確認
- ボーレートが正しいか確認（通常9600）
- 重量計の電源を確認
